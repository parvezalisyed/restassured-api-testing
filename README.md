# RestAssured API Testing Suite

Automated testing for 25+ endpoints using public APIs:
- ReqRes.in (GET, POST, PUT, DELETE users)
- JSONPlaceholder (posts, comments)

Features
- Positive & negative test cases
- JSON response validation
- Authentication testing
- Schema validation

Tech Stack
- Java
- RestAssured
- TestNG
- Maven

How to Run
```bash
mvn clean test
